# Features
* 🈚 Free w/no ads
* 🖨️ Print-friendly
* 🗝️ Answer key on last page.
* 📝 Many worksheet types, all randomized for near-infinite iterations
* 👨‍💻 Open source
* 🔢 1-digit equations are all possible equations not including 1.
* 😀 Emojis using [Twemoji](https://twemoji.twitter.com)
